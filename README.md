# Atharva Pandkar Portfolio (Fresh Start)

This repository is the root GitHub repository and now uses a clean React + Vite + TypeScript setup.

## Development


- `npm install`
- `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Structure

- `archive/legacy-portfolio/` legacy portfolio snapshots and notes
- `src/content/` centralized personal content
- `src/assets/` assets and documents
- `src/lib/analytics.ts` PostHog analytics abstraction
- `src/analytics/events.ts` event taxonomy
